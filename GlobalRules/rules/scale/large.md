# Applied Rules: 🔴 Large Scale

> Many files, multiple modules, released or ongoing development

## Apply All Rules

Apply rules progressively:

1. Core rules
2. Language rules
3. Relevant detail rules only
4. Load additional rules only when needed
Do not load all rules into context at once
Only exclusions need to be stated in PROJECT_CONFIG.md with reasons.

## Large-Scale Specific Notes

- Consider index.yaml splitting when it grows large (per-directory)
- Security clearance is mandatory
- Keep ai_log.md updated in real time
- Keep CONTEXT.md current at all times
- Pay special attention to the retreat rule (Section 24-4)
- Context summarization and disposal (Section 24-5) is critical
